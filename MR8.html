例題 ─ 2桁の整数の問題
十の位・一の位を文字でおく / 中学2年
例題
2桁の自然数がある。その数は各位の数の和が 7 で，十の位の数と一の位の数を入れかえてできる数は，もとの数より 27大きく なるという。もとの自然数を求めなさい。
📐 2桁の整数の表し方
2桁の整数の表し方 📌 もとの数の 十の位を x、一の位を y とおく
📌 もとの数 = 10x + y
📌 入れかえた数 = 10y + x
📌 注意：「x + y」は位の数の和、「10x + y」は整数の値

STEP 1　x、y をおいて整理する

もとの数の十の位を x、一の位を y とおく。

もとの数 = 10x + y 入れかえた数 = 10y + x

【条件1】各位の数の和が7 → x + y = 7 【条件2】入れかえた数 = もとの数 + 27 → 10y + x = 10x + y + 27

STEP 2　連立方程式をつくる

① 位の和：　x + y = 7 ② 入れかえの関係：　10y + x = 10x + y + 27

②を整理：左辺に文字、右辺に数字を集める

10y + x − 10x − y = 27 −9x + 9y = 27 両辺を9で割って −x + y = 3 ……②'

STEP 3　連立方程式を解く

①＋②' で x を消す：

x + y = 7 ……① +) −x + y = 3 ……②' ─────────────── 2y = 10 ∴ y = 5

①に y = 5 を代入：

x + 5 = 7 x = 2

STEP 4　答えと確かめ

もとの数 = 10×2 + 5 = 25

【確かめ】 　位の和：2 + 5 = 7 ✓ 　入れかえた数：52 　52 − 25 = 27 ✓
