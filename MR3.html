例題 ─ 時間と距離（往復・峠越え）
行きと帰りで上り・下りが入れ替わる問題 ／ 中学2年
例題
A町から峠を越えてB町までの道のりについて，バスでA町から峠を越えてB町まで 15分 かかった。同じ道をB町から峠を越えてA町まで 13分 かかった。バスは上りは時速 30km，下りは時速 40km という。A町から峠までの距離，峠からB町までの距離をそれぞれ求めよ。
📐 線分図（行き：A→B　／　帰り：B→A）
往復・峠越えのコツ 📌 行きと帰りで「上り」と「下り」が入れ替わること！
📌 距離 x, y は同じ（同じ道を通る）が、速さが入れ替わるので時間が変わる
📌 行きの時間と帰りの時間で 2つの式 をつくる

STEP 1　x、y をおいて表を作る

A町から峠までを x km、峠からB町までを y km とおく。 15分 = 1560 = 14 時間 13分 = 1360 時間

区間速さ距離時間
行き
(A→B)		A→峠（上り）30 km/時xx30
峠→B（下り）40 km/時yy40
帰り
(B→A)		B→峠（上り）30 km/時yy30
峠→A（下り）40 km/時xx40

STEP 2　連立方程式をつくる

① 行きの時間：　x30 + y40 = 1560 ② 帰りの時間：　x40 + y30 = 1360

STEP 3　両方とも分母をはらう（両辺×120）

①×120：

x y 15 ─── + ─── = ─── の両辺に 120 をかける 30 40 60 120×x/30 + 120×y/40 = 120×15/60 4x + 3y = 30 ……①'

②×120：

x y 13 ─── + ─── = ─── の両辺に 120 をかける 40 30 60 120×x/40 + 120×y/30 = 120×13/60 3x + 4y = 26 ……②'

x を消すために、①'×3 と ②'×4 を引く：

①'×3：　12x + 9y = 90 ②'×4：　12x + 16y = 104 ②'×4 − ①'×3 12x + 16y = 104 −) 12x + 9y = 90 ─────────────────── 7y = 14 ∴ y = 2

①'に y = 2 を代入：

4x + 3×2 = 30 4x + 6 = 30 4x = 24 x = 6

STEP 4　答えと確かめ

A町から峠まで 6 km，峠からB町まで 2 km

【確かめ】 　行き：6÷30 + 2÷40 = 0.2 + 0.05 = 0.25時間 = 15分 ✓ 　帰り：6÷40 + 2÷30 = 0.15 + 0.0667 ≒ 0.2167時間 ≒ 13分 ✓

求めるのは 速さ！上りの時速を x km/時、下りの時速を y km/時 とおく。「距離＝速さ×時間」で、行きの合計距離と帰りの合計距離はどちらも36km。20分=1/3時間、30分=1/2時間、40分=2/3時間、15分=1/4時間。

段階解説

上りの時速を x km/時、下りの時速を y km/時 とおく。

分を時間にする： 20分 = 2060 = 13 時間 30分 = 3060 = 12 時間 40分 = 4060 = 23 時間 15分 = 1560 = 14 時間

「距離＝速さ×時間」 を使うのがポイント。

STEP 2　連立方程式をつくる

行き：A→峠（上り、1/3時間） と 峠→B（下り、1/2時間）の合計が36km 帰り：B→峠（上り、2/3時間） と 峠→A（下り、1/4時間）の合計が36km

① 行きの距離：　13x + 12y = 36 ② 帰りの距離：　23x + 14y = 36

STEP 3　両方とも分母をはらう

①×6：

1 1 ─── x + ─── y = 36 の両辺に 6 をかける 3 2 2x + 3y = 216 ……①'

②×12：

2 1 ─── x + ─── y = 36 の両辺に 12 をかける 3 4 8x + 3y = 432 ……②'

②' − ①' で y を消す（係数が同じ）：

8x + 3y = 432 ……②' −) 2x + 3y = 216 ……①' ────────────────── 6x = 216 ∴ x = 36

①'に x = 36 を代入：

2×36 + 3y = 216 72 + 3y = 216 3y = 144 y = 48
